# Collaboration

- I'm a solo founder building tariffa with AI agents as my pair. The code has to be readable and maintainable by me coming back cold, and by an agent picking up a task with no prior context: clear naming, modular structure, no hidden side effects, well-defined interfaces.
- Write features so they can be lifted out later: generic interfaces, minimal coupling to the rest of the project.
- Optimise for the next session over cleverness now. If a future me (or a future agent) would have to re-derive why something works, leave a short note instead.
